## Usage

- Function `safeMint()` only callable by the owner can mints one new NFTs with an auto-increments ids, you'll see 
- Check ownership of the NFT, it returns the current wallet address that owns the given token.
- Function `tokenURI` returns the metadata URL from IPFS, it can be opened on a browser with ipfs.io/ link.
- We can view `NFT` on the wallet after minting.

## Other usages 

- Utility token in a DApp → e.g., spend tokens to unlock content.
- Reward system → reward users for actions (like points).
- Voting token → each token = one vote in governance.
- Test environment → developers can use it to practice ERC-20 integration.
